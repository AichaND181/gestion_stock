# Gestion de Stock

Application web de gestion d'inventaire développée avec PHP et MySQL.

## Fonctionnalités

- 🔐 Authentification (login / mot de passe)
- 📊 Tableau de bord avec indicateurs
- 📈 Graphiques des ventes par catégorie
- 📦 Gestion des produits, fournisseurs, catégories, clients

## Technologies

- PHP
- MySQL
- HTML / CSS
- JavaScript
